1. How to Run
-------------
	The Github repository is provided here, but in case the project doesn't run when downloaded,
	you can follow the other steps to get the project running.
	https://github.com/tfang7/RLfighter
	^-- import this as a new file system into a new Java project with the same name: "FightingAI"
	
 1. Install JDK/Eclipse
	http://www.ice.ci.ritsumei.ac.jp/~ftgaic/index-2.html
	
	
 2. AIToolKit Installation
	- This is used to start up a new AI project.
	http://www.ice.ci.ritsumei.ac.jp/~ftgaic/index-2h.html

 3. AI Simulation Installation
	http://www.ice.ci.ritsumei.ac.jp/~ftgaic/index-2h.html
	- Follow the instructions in the under the section "Test the AI code"
	
 4. Export the AI project as a JAR file to the ai/data folder in the AI Simulation project
